    class SyntaxTree::Rake::WriteTask

    A Rake task that runs format on a set of source files.

    Example:

    require 'syntax_tree/rake/write_task'

SyntaxTree::Rake::WriteTask.new do |t|
  t.source_files = '{app,config,lib}/**/*.rb'
end

    This will create task that can be run with:

    rake stree_write

    Attributes

    name[RW]

    Name of the task. Defaults to :“stree:write”.

    plugins[RW]

    The set of plugins to require. Defaults to [].

    source_files[RW]

    Glob pattern to match source files. Defaults to ‘lib/*/.rb’.
